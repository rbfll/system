# 유닉스 역사 및 표준

## AT&T 벨 연구소(Bell Lab)에서 개발됨
- **Ken Thompson**이 어셈블리어로 개발
- **D. Ritchie**가 C 언어로 다시 작성  
  - C 언어는 Unix를 작성하기 위한 언어로 밀접하게 관련됨
  - 이론적으로 C 컴파일러만 있으면 이식 가능
- 소스코드를 대학에 개방함

## 유닉스의 큰 흐름
- **System V (시스템 V)**
- **BSD (Berkeley Standard Distribution) 유닉스**
- **리눅스 (Linux)**
